什么是Git

  • 解决代码的丢失和备份问题

  • Git 是一个版本控制系统，可以让你跟踪你对文件所做的修改

    • 例如学习一个项目，跟着敲代码，如果全程提交的话，也方便自己复盘

  • 与GitHub / Gitee 配合使用
