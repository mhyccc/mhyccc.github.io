问题

Web server failed to start. Port 8080 was already in use.

解决方法

linux

  1. 查找正在使用端口 8080 的进程 ID
  1. 使用上面结果中的进程 ID 杀死进程

windows

以管理员身份打开终端

  1. 查找进程ID(process_id)
  1. 根据进程ID终止进程
